# isic-rss

RSS module for the [Discord](https://discordapp.com) bot [isic](https://github.com/atomicptr/isic).

## Usage

### Add new RSS feeds

via ```@BOT_NAME rss add RSS_URL```

### Remove RSS feeds

via ```@BOT_NAME rss remove RSS_URL```

### List your RSS feeds

via ```@BOT_NAME rss list```
